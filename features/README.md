# Features Klasörü

Bu klasör, uygulamanın temel işlevsel kaynak kodlarını ve farklı özellik parçalarını (bileşenlerini) barındırmak için ayrılmıştır. Mevcut proje mimarisinde bileşenler ve sayfalar kök dizindeki klasörlerde (`Egitimler`, `Alıştırmalar`, `Başarılarım`) ve kendi HTML/CSS yapılarında tutulmaktadır.
İlerleyen aşamalarda modüler JavaScript mantığı (modüller, modeller, view kontrolcüleri) bu klasör altına taşınabilir.
